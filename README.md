# dom-inspector-widget

a DOM inspector widget. click to activate, then all elements on page can be hovered to highlight/outline the element, right click to show custom context menu on element that exposes button to save the DOM/Element by copy the elements html and posting it to server.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/dom-inspector-widget.git
cd dom-inspector-widget
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
